# Kisan Sethu | Eligibility Verified Portal

**Kisan Sethu** is a precision eligibility verification platform designed to match farmers with state-specific agricultural schemes. 

## 🚀 Features
* **Multilingual Support:** Interface available in English, Hindi (हिन्दी), and Telugu (తెలుగు).
* **Smart Matching:** Filters schemes based on State, Crop, Land size, and Income range.
* **Probability Indicator:** Calculates the success probability of a scheme application based on user profile.
* **Deadline Alerts:** Displays important upcoming dates for scheme registrations.
* **OTP Verification:** Includes a simulated 2-step verification process for notifications.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Responsive Design)
* **Logic:** Vanilla JavaScript

## 📖 How to Use
1. Clone the repository or download the `Kisan sethu.html` file.
2. Open `Kisan sethu.html` in any modern web browser.
3. Select your language and fill in your farming profile to see eligible schemes.
